Aula 10: Criando Elementos

Como gerar HTML dinamicamente usando JavaScript.

1. Definições Principais

Até agora, mudámos o que já estava lá. Hoje, vamos criar o que não existe:

  • document.createElement('tag'): Cria um elemento na memória (ex: uma div ou h1).
  • appendChild(): Insere o elemento criado dentro de outro elemento "pai" na página.

2. Exemplo Prático

Clique no botão para fabricar um novo card de produto no contentor abaixo.

3. Código do Exemplo (HTML + CSS + JS)

<!-- HTML: Onde o item vai morar -->
<div id="lista"></div>
<button onclick="criar()">Adicionar</button>

/* CSS: Estilo do novo item */
.item { background: white; border: 1px solid purple; padding: 10px; }

// JAVASCRIPT: A lógica da fábrica
function criar() {
  // 1. Seleciona o pai
  const pai = document.getElementById('lista');

  // 2. Cria o novo elemento na memória
  const novoItem = document.createElement('div');

  // 3. Configura (texto e classe)
  novoItem.innerText = "Novo Produto";
  novoItem.classList.add('item');

  // 4. Pendura o filho no pai
  pai.appendChild(novoItem);
}

4. Atividade de Fixação

Modifique o código acima para realizar as seguintes tarefas:

  1. Tarefa 1: Faça com que cada novo item tenha um número diferente (Ex: Produto 1, Produto 2).
  2. Tarefa 2: Adicione uma linha para que o fundo do novo item seja amarelo claro (#ffffcc).
  3. Tarefa 3: Use innerHTML em vez de innerText para adicionar um ícone de estrela ⭐ ao lado do nome do produto.
  4. Tarefa 4: Crie um botão "Remover" que limpe apenas o último item adicionado.
  5. Tarefa 5: Adicione um efeito de borda vermelha se o número do produto for par.
