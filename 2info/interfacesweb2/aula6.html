Aula 06: O DOM e a Seleção de Elementos

Como o JavaScript encontra o que precisa no HTML.

1. O que é o DOM?

O DOM (Document Object Model) é uma representação da sua página HTML como se fosse uma árvore de objetos. Cada tag (h1, div, p) é um "nó" que o JavaScript pode agarrar e modificar.

Existem duas formas principais de selecionar elementos hoje em dia:

  • getElementById: Busca um elemento pelo seu ID único. É muito rápido.
  • querySelector: O seletor moderno. Usa a mesma sintaxe do CSS ( .classe, #id, tag ).

3. Código Exemplo (Seleção Completa)

Estude como as variáveis guardam a referência do elemento HTML:

<!-- Estrutura HTML -->
<h1 id="titulo">Olá Mundo</h1>
<p class="descricao">Texto de exemplo.</p>

/* Estilo CSS */
.destaque { border: 2px solid blue; padding: 10px; }

// JavaScript: Selecionando Elementos
function manipularElementos() {
    // 1. Selecionando pelo ID (Antigo mas eficiente)
    const elementoId = document.getElementById('titulo');

    // 2. Selecionando pela Classe usando Query Selector (Moderno)
    // Nota: No querySelector usamos o ponto (.) para classes
    const elementoClasse = document.querySelector('.descricao');

    // 3. Aplicando mudanças para provar que selecionou
    elementoId.style.color = "blue";
    elementoClasse.classList.add('destaque');
}
