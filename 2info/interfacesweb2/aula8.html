Aula 08: Estilização em Tempo Real

Dominando a propriedade .style para criar interfaces vivas.

1. A Propriedade .style

Através do JavaScript, podemos aceder a qualquer propriedade CSS de um elemento. A regra principal é a conversão de nomes para camelCase:

  • CSS: background-color   ➜   JS: backgroundColor
  • CSS: font-size   ➜   JS: fontSize
  • CSS: border-radius   ➜   JS: borderRadius

Os valores devem ser sempre Strings (entre aspas) e incluir unidades quando necessário (ex: "20px").

3. Código Exemplo (Ficheiro Completo)

Abaixo está o exemplo completo integrando HTML, CSS e JS. Copia este modelo para testar no teu computador:

<!DOCTYPE html>
<html lang="pt-pt">
<head>
    <style>
        /* CSS: Definimos a base visual */
        #caixa {
            width: 100px;
            height: 100px;
            background-color: grey;
            transition: 0.5s; /* Suaviza a mudança */
        }
    </style>
</head>
<body>

    <div id="caixa"></div>
    <button onclick="aplicarEstilo()">Mudar Visual</button>

    <script>
        function aplicarEstilo() {
            // 1. Selecionamos o elemento
            const el = document.getElementById('caixa');
            
            // 2. Aplicamos estilos diretamente (Nota o camelCase)
            el.style.backgroundColor = "blue";
            el.style.width = "200px";
            el.style.borderRadius = "20px";
            el.style.border = "5px solid orange";
        }
    </script>

</body>
</html>
