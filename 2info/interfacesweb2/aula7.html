Aula 07: Manipulando Conteúdos

Aprendendo a diferença entre texto puro e estruturas HTML dinâmicas.

1. innerText vs innerHTML

Após selecionar um elemento, podemos alterar o que está dentro dele de duas formas principais:

  • innerText: Altera apenas o texto puro. Se escreveres tags, elas aparecem como texto comum.
  • innerHTML: Interpreta o código HTML. Permite criar novos elementos e estilos dinamicamente.

Basicamente: innerText é para palavras; innerHTML é para componentes e tags.

3. Código Exemplo (Ficheiro Completo)

Este código abaixo integra as três linguagens. Copia este modelo para entender como o processo funciona:

<!DOCTYPE html>
<html lang="pt-pt">
<body>

    <!-- Estrutura HTML -->
    <div id="area">Texto Original</div>
    <button onclick="atualizar()">Mudar Tudo</button>

    <style>
        /* CSS: Definimos o visual básico */
        #area { padding: 20px; border: 1px solid black; }
        .destaque { color: red; font-weight: bold; }
    </style>

    <script>
        function atualizar() {
            // 1. Selecionamos o elemento
            const elemento = document.getElementById('area');

            // 2. Usamos innerHTML para injetar uma estrutura nova
            elemento.innerHTML = `
                <h2 class="destaque">Novo Título</h2>
                <p>Este parágrafo foi criado via JS!</p>
            `;
            
            // 3. Exemplo de innerText (apenas texto)
            // elemento.innerText = "Apenas texto aqui";
        }
    </script>

</body>
</html>
