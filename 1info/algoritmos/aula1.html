Aula 01: Introdução e Saída de Dados

Nesta primeira aula, você aprenderá como o computador processa informações e como fazê-lo exibir suas primeiras mensagens na tela.

1. Definições Principais

O que é Algoritmo?

É um conjunto finito de passos organizados para atingir um objetivo. Como uma receita de bolo: se pular o passo de bater os ovos, o bolo não cresce.

Lógica de Programação

É a técnica de encadear pensamentos para atingir a solução. O computador é rápido, mas "burro"; você precisa dar instruções exatas.

Sintaxe do Visualg

É o conjunto de regras de escrita. O código sempre começa com algoritmo e termina com fimalgoritmo.

2. Exemplos Práticos Comentados

A Diferença entre Escreva e Escreval 

algoritmo "Exemplo_Quebra_Linha"

inicio
   // O comando 'escreva' NÃO pula linha. O próximo texto ficará grudado.
   escreva("Olá, ")
   escreva("Mundo!")
   
   // O comando 'escreval' pula para a linha de baixo após escrever.
   escreval("") // Apenas pula uma linha vazia
   escreval("Eu estou na linha 2.")
   escreval("Eu estou na linha 3.")
   
fimalgoritmo

B Criando Interfaces Visuais 

algoritmo "Layout_Console"

inicio
   // Usamos caracteres como '*' e '-' para organizar a tela
   escreval("=============================")
   escreval("   LOGIN DO SISTEMA - 2026   ")
   escreval("=============================")
   escreval("")
   escreval("Usuário: [DIGITE AQUI]")
   escreval("Senha  : [********]")
   escreval("")
   escreval("-----------------------------")
fimalgoritmo

C Sequenciamento de Passos (Logística) 

algoritmo "Passo_a_Passo"

inicio
   // A ordem das instruções importa! 
   escreval("ROTINA MATINAL:")
   escreval("1. O despertador toca às 06:00.")
   escreval("2. Abrir os olhos e bocejar.")
   escreval("3. Levantar da cama.")
   escreval("4. Escovar os dentes.")
   escreval("5. Tomar café reforçado.")
   escreval("---")
   escreval("Se invertermos o 3 e o 2, você levanta de olho fechado!")
fimalgoritmo

3. Exercícios de Fixação

01

Primeiras Palavras: Desenvolva um algoritmo que apresente sua identidade ao computador. Use o comando escreval para exibir seu nome completo em uma linha e, na linha seguinte, a mensagem: "Olá mundo, estou iniciando minha jornada na programação!"

02

Localização Geográfica: Crie um programa que informe onde você está estudando. Exiba o nome da sua cidade na primeira linha e o seu estado na segunda linha. Regra: Você deve garantir que as duas informações não fiquem na mesma linha do console.

03

Arte Digital Básica: Utilize seus conhecimentos de saída de dados para desenhar um quadrado sólido no console. Ele deve ter 4 linhas e 4 colunas, todo preenchido com a letra "X". Tente manter o alinhamento visual perfeito.

04

Lista de Tarefas: Imagine que você precisa organizar seu dia. Escreva um algoritmo que exiba uma lista com 4 tarefas (ex: 1. Acordar, 2. Estudar...). Cada tarefa deve aparecer em uma linha diferente e numerada, facilitando a leitura.

05

Ficha de Perfil: Simule a exibição de um "Card de Usuário". O programa deve mostrar quatro categorias fixas: NOME:, IDADE:, CIDADE: e HOBBY:. Preencha com seus dados reais logo à frente das categorias, organizando uma por linha.

06

Cabeçalho de Tabela: Desenvolva a estrutura superior de um relatório de estoque. A saída deve exibir exatamente assim: | PRODUTO | QTD | PREÇO |. Abaixo dessa linha, crie uma divisória usando hífens (---) para separar o título dos dados.

07

Lógica Sequencial: Escreva o passo a passo lógico para a tarefa "Fazer um Miojo". Você deve usar pelo menos 6 linhas de comando, descrevendo desde colocar a água no fogo até servir no prato. A ordem das instruções deve ser coerente.

08

O Desafio do Espaçamento: Use apenas o comando escreva (sem o "L" final) quatro vezes seguidas para montar a frase "EU GOSTO DE ALGORITMOS". Atenção: Como você vai garantir que as palavras não fiquem grudadas umas nas outras?

09

Geometria no Console: Tente desenhar um triângulo (pirâmide) simples no centro da tela. Use barras inclinadas / \ para as laterais e o caractere underline _ para fechar a base. Use espaços em branco para centralizar o topo.

10

Desafio Master: Crie um "Cupom Fiscal de Mercado". O topo deve ter o nome da "LOJA ALGORITMOS 2026", seguido por uma linha tracejada. Liste 2 produtos com preços e, no final, exiba o "TOTAL" dentro de uma moldura de asteriscos.
