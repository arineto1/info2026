Tabelas Estruturadas e Atributos Globais

O Arquivo e os Crachás

Organizar um site é como gerir um Arquivo de Escritório. Precisamos de gavetas para os dados e crachás para identificar os elementos.

Tabelas: As Gavetas

Imagine uma planilha financeira. As tabelas HTML organizam dados em linhas e colunas. Usamos semântica (Topo, Corpo e Base) para que o sistema entenda o que é um título e o que é o resultado final.

ID e Classe: Os Crachás

O id é um crachá individual e único (como um Cartão de Cidadão). A class é o uniforme da equipa (vários elementos podem usar a mesma classe para receberem o mesmo estilo).

1. Anatomia das Tabelas e Identidade

<thead>, <tbody>, <tfoot>

O trio de ouro das tabelas. Divide a informação em Cabeçalho (títulos), Corpo (dados reais) e Rodapé (totais ou conclusões).

<th> vs <td>

<th> define uma célula de cabeçalho (geralmente em negrito). <td> define uma célula de dado comum.

ID (Identificador Único)

O atributo id deve ser único na página. É usado para links internos e para aplicar estilos específicos a um só elemento.

CLASS (Classificação)

O atributo class pode ser repetido em vários elementos. É a forma mais eficiente de agrupar elementos que devem parecer iguais.

2. Prática: O Relatório de Vendas

O código abaixo demonstra uma tabela financeira moderna usando atributos globais para organizar a estrutura. Estude os comentários:

<!DOCTYPE html>
<html lang="pt-pt">
<head>
    <meta charset="UTF-8">
    <title>Aula 04 - Tabelas e Atributos</title>
</head>
<body>

    <!-- Atributo ID: Define um identificador único para este título -->
    <h1 id="main-title">Relatório Financeiro Q1</h1>

    <!-- Atributo CLASS: Agrupa este parágrafo numa categoria de 'destaque' -->
    <p class="text-highlight">Abaixo apresentamos os dados oficiais do primeiro trimestre.</p>

    <!-- Tabela com borda para visualização rápida (border=1 é obsoleto, mas ajuda no estudo) -->
    <table border="1" id="sales-table">
        
        <!-- thead: Define o cabeçalho fixo da tabela -->
        <thead>
            <tr>
                <!-- th: Célula de cabeçalho (título da coluna) -->
                <th>Produto</th>
                <th>Quantidade</th>
                <th>Preço Unitário</th>
                <th>Total</th>
            </tr>
        </thead>

        <!-- tbody: Onde os dados reais são inseridos -->
        <tbody>
            <!-- Linha 1 -->
            <tr>
                <td>Notebook Pro 2026</td>
                <td>10</td>
                <td>1.200,00€</td>
                <td>12.000,00€</td>
            </tr>
            <!-- Linha 2 -->
            <tr>
                <td>Monitor 4K Curved</td>
                <td>05</td>
                <td>450,00€</td>
                <td>2.250,00€</td>
            </tr>
        </tbody>

        <!-- tfoot: Rodapé para somatórios ou notas -->
        <tfoot>
            <tr>
                <!-- colspan: Faz esta célula ocupar o espaço de 3 colunas -->
                <td colspan="3"><strong>Faturamento Bruto:</strong></td>
                <td><strong>14.250,00€</strong></td>
            </tr>
        </tfoot>
    </table>

    <!-- Atributo 'title': Mostra uma dica ao passar o rato (mouse) -->
    <p title="Última atualização: 18 de Fevereiro de 2026">
        Passe o rato aqui para ver a data da última revisão.
    </p>

</body>
</html>

3. Atividade: O Gestor de Dados

Desafie-se a expandir a inteligência deste documento:

1

Adicionar Dados: Insira uma terceira linha no tbody para o produto "Teclado Mecânico RGB", quantidade 20, preço 80€ cada.

2

Identidade Única: Atribua um id="footer-note" ao parágrafo final (o que tem o atributo title).

3

Classificação em Massa: Dê a mesma class="row-item" para todas as linhas (<tr>) que estão dentro do tbody.

4

Mesclagem de Colunas: Altere o colspan do rodapé para "2" e veja como a tabela se desorganiza. Depois corrija adicionando uma célula vazia para equilibrar.

5

Aviso Global: Adicione o atributo lang="en" apenas na célula que contém "Notebook Pro" para indicar ao sistema que aquele termo específico está em inglês.
