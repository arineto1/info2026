Aula 01: Setup e Saída de Dados

Aprenda a configurar o seu ambiente de trabalho e a dominar a primeira forma de comunicação entre o software e o utilizador.

1. Instruções: Como Preparar o Ambiente

Siga estes passos para configurar o VS Code profissionalmente.

01

Instalar Extensões

Abra o VS Code, pressione Ctrl+Shift+X e procure por Python (da Microsoft) e Pylance. Clique em "Install". Estas extensões dão inteligência ao editor.

02

Ativar o Code Runner

Instale a extensão Code Runner. Ela adicionará um ícone de "Play" no canto superior direito do seu ecrã, permitindo executar o código com apenas um clique.

03

Configurar Terminal Integrado

Para garantir que o código funcione no terminal interativo: Vá a File > Preferences > Settings, procure por "Code-runner: Run In Terminal" e marque a caixa de seleção.

04

Criar o Primeiro Ficheiro

Clique em File > New File e guarde-o com o nome aula01.py. O sufixo .py é obrigatório para que o sistema reconheça como um script Python.

2. Fundamentos do Sistema

Conceitos essenciais para compreender a arquitetura do Python.

Python Interpretado

Ao contrário de linguagens compiladas, o Python executa o código linha a linha. Isto torna o desenvolvimento desktop mais rápido e flexível.

O Interpretador

É o motor que traduz o teu ficheiro .py em linguagem que o processador entende. Sem ele, o código é apenas texto.

Sintaxe e Indentação

No Python, o espaço vazio (indentação) é obrigatório. Ele define onde começa e termina cada bloco de lógica do teu software.

Console (Output)

A função print() é a tua ponte. É através dela que o programa envia feedback visual para o utilizador.

3. Prática: O Painel do Shinobi

Bloco bloqueado: Digite manualmente no seu VS Code para praticar a sintaxe ninja!

REDE_KONOHA.PY 
# --- CABEÇALHO DA ALDEIA DA FOLHA ---
# O sinal '*' cria uma moldura inspirada em pergaminhos ninja
print("@" * 50)
print("REDE DE INTELIGÊNCIA SHINOBI // KONOHAGAKURE")
print("SISTEMA DE MONITORIZAÇÃO DE CHAKRA v7.0")
print("@" * 50)

# --- LOGS DE ACESSO (Simulação de ativação de Jutsu) ---
print("\nA sincronizar Chakra com o Pergaminho...", end=" ")
print("[100%]")

print("A verificar selos de proteção da barreira...", end=" ")
print("[ATIVO]")

# --- TABELA DE NINJAS (Uso de 'sep' para dados de equipa) ---
print("\n" + "-" * 15 + " EQUIPA 7 ATUAL " + "-" * 15)
print("Ninja", "Rank", "Especialidade", sep=" | ")
print("Naruto", "Hokage", "Rasengan", sep=" | ")
print("Sasuke", "Suporte", "Chidori", sep=" | ")

# --- FORMATAÇÃO COM ESCAPE (\n e \t) ---
print("\nMISSÕES DISPONÍVEIS NO QUADRO:")
print("\t* Rank S: Detetar rasto da Akatsuki")
print("\t* Rank B: Escolta para a Aldeia da Areia")

# --- OPERAÇÕES MATEMÁTICAS DIRETAS NO PRINT ---
print("\nEstatísticas da Batalha Anterior:")
print("Total de Chakra consumido (unidades):", 5000 + 2350)
print("Média de Kunais por Shinobi:", 60 / 3)

# --- MENSAGEM FINAL ---
print("\n" + "@" * 50)
print("KONOHA ESTÁ SEGURA. SISTEMA EM REPOUSO.")
print("@" * 50)

4. Atividade Prática

Resolva os exercícios para consolidar o conhecimento.

Nível: Fácil

01. Crie um script que exiba o seu nome e a sua cidade em linhas separadas.

02. Imprima a frase "Python*é*Incrível" usando obrigatoriamente o argumento sep.

03. Exiba o resultado da soma de 1500 + 3500 sem usar aspas no cálculo.

Nível: Médio

04. Etiqueta de Produto: Use \n e \t para criar uma etiqueta com "PRODUTO", "CÓDIGO" e "VALOR" perfeitamente alinhados.

05. Loading Bar: Simule o carregamento de uma barra usando "#" * 10 e a palavra "FIM" na mesma linha usando end.

Nível: Desafio

06. Simulador de Cupom Fiscal

Crie um script que gere um cupom de venda. Deve conter:

  • Divisórias horizontais usando "-" * 35.
  • Três itens (Ex: Mouse, Teclado, Monitor).
  • Preços alinhados à direita usando o caractere \t.
  • Uma linha de "TOTAL" onde o valor é calculado automaticamente pela soma dos itens dentro do próprio print.
