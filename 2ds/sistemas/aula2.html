Aula 02: Variáveis e Tipos

Aprenda a armazenar informações na memória do J.A.R.V.I.S. e manipular os dados essenciais para a missão.

1. O Arquivo de Dados

Como o computador guarda a identidade e os poderes.

O que é uma Variável?

Pense numa variável como uma caixa etiquetada nas Indústrias Stark. Você cria a etiqueta (nome) e guarda algo dentro (valor). Sempre que precisar desse valor, chama pelo nome da etiqueta.

Tipagem Dinâmica

No Python, o sistema adapta-se automaticamente. Se guardar texto, a caixa vira tipo Texto. Se guardar números, vira tipo Número. Não precisa declarar antes.

str

String (Texto)

"Iron Man"

int

Inteiro

3000

float

Flutuante (Decimal)

98.5

bool

Booleano

True / False

2. Prática: Protocolo J.A.R.V.I.S.

Digite este código para inicializar os sistemas da armadura Mark 85.

MARK_85_SYSTEM.PY 
# --- DEFINIÇÃO DE VARIÁVEIS (MEMÓRIA PRINCIPAL) ---
# Strings (str): Textos devem estar sempre entre aspas
nome_piloto = "Tony Stark"
codinome = "Iron Man"
filiacao = "Vingadores"

# Inteiros (int): Números para contagem e versões
versao_armadura = 85
municao_repulsor = 500

# Floats (float): Números decimais para precisão de energia
nivel_bateria = 98.5
consumo_por_minuto = 2.5

# Booleanos (bool): Interruptores lógicos (True/False)
sistemas_online = True
ameaca_detectada = False

# --- INTERFACE DE COMANDO (HUD) ---
print("--- INICIANDO J.A.R.V.I.S ---")
print("Bem-vindo,", nome_piloto)
print("Armadura Ativa: Mark", versao_armadura)

# Diagnóstico de Tipos: Verificando como o Python guardou os dados
print("\n[DIAGNÓSTICO DE MEMÓRIA]")
print("Variável 'bateria' é do tipo:", type(nivel_bateria))
print("Variável 'online' é do tipo:", type(sistemas_online))

# --- CÁLCULOS DE MISSÃO ---
# Usando variáveis para criar novas informações
tempo_voo_restante = nivel_bateria / consumo_por_minuto

print("\n[ESTIMATIVA TÁTICA]")
print("Tempo de voo disponível:", tempo_voo_restante, "minutos")

# --- SIMULAÇÃO DE COMBATE ---
print("\n>>> ALERTA: DISPARANDO UNI-BEAM <<<")
# Atualizando o valor de uma variável existente
nivel_bateria = nivel_bateria - 15.5
municao_repulsor = municao_repulsor - 50

print("Status Pós-Combate:")
print("Bateria:", nivel_bateria, "%")
print("Munição:", municao_repulsor, "unidades")
